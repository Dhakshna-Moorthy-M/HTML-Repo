A WEB DEVELOPER

A Web Developer is in charge of ensuring websites look good and function properly. Web developers collaborate with website and graphic designers, monitor website traffic,troubleshoot website problems when they arise, and update websites as necessary.

HTML/CSS. To work as a web developer, you must be familiar with HTML and CSS, JavaScript., Git.Backend Tech. JS Libraries and Frameworks.Testing and Debugging. Search Engine Optimization(SEO) Skills.Responsive Web Designing Skills.

Documents and downloadable media are made available to the network through web servers and can be accessed by programs such as web browsers. Servers and resources on the World Wide Web are identified and located through character strings called uniform resource locators (URLs).
